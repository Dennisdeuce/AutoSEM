AutoSEM Orchestrator

Automated SEM Campaign Engine for Court Sportswear

Overview

AutoSEM is a fully automated search engine marketing orchestrator that manages Google Ads and Meta advertising campaigns for Court Sportswear's print-on-demand tennis apparel business.

Architecture

Core Stack

  • Backend: Python / FastAPI
  • Database: PostgreSQL / SQLite
  • ORM: SQLAlchemy
  • Deployment: Docker / Replit
  • Integrations: Shopify API, Google Ads API, Meta Marketing API

Safety Mechanisms

  • Daily Spend Limit: $200/day
  • Monthly Spend Limit: $5,000/month
  • ROAS Threshold: 1.5x minimum
  • Emergency Pause: $500 net loss
